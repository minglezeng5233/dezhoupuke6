# 德州扑克3 - 专业手游版

基于Kivy框架的移动端优化德州扑克游戏，支持Android APK构建。

## 🎮 游戏特色

### 1. **移动端优化**
- 专为手机触摸操作设计
- 720x1280竖屏分辨率适配
- 触摸友好的按钮和界面
- 响应式布局适配不同屏幕

### 2. **完整游戏功能**
- 5人桌德州扑克（4AI + 1人类）
- 完整游戏流程：翻牌前→翻牌→转牌→河牌→摊牌
- 智能AI对手系统
- 自动盲注和下注管理

### 3. **专业界面设计**
- 绿色椭圆牌桌
- 彩色玩家卡片围绕牌桌
- 实时游戏状态显示
- 精美的卡牌渲染效果

### 4. **技术架构**
- 基于Kivy 2.2.1框架
- 支持Android APK构建
- 模块化代码结构
- 性能优化和内存管理

## 📱 构建和安装

### 前置要求
- Python 3.6+
- Buildozer
- Android SDK/NDK (自动下载)

### 构建APK

#### 方法一：使用本地构建脚本（推荐）
```bash
# 给脚本执行权限
chmod +x local_build.sh

# 运行构建
./local_build.sh
```

#### 方法二：手动构建
```bash
# 清理构建缓存
buildozer android clean

# 构建调试版本
buildozer android debug

# 构建发布版本
buildozer android release
```

### 安装到手机
1. 将生成的APK文件传输到Android设备
2. 在设置中允许"未知来源"安装
3. 使用文件管理器安装APK
4. 运行德州扑克3应用

## 🎯 游戏操作

### 操作按钮
- **弃牌**: 放弃当前手牌
- **过牌**: 不下注，保持当前状态
- **跟注**: 匹配当前下注金额
- **加注**: 增加下注金额
- **全下**: 押上所有筹码

### 游戏流程
1. **盲注阶段**: 前两位玩家下盲注
2. **翻牌前**: 玩家根据手牌强度行动
3. **翻牌**: 发出3张公共牌
4. **转牌**: 发出第4张公共牌
5. **河牌**: 发出第5张公共牌
6. **摊牌**: 未弃牌玩家比较手牌

## 🏆 AI系统

### AI特性
- **随机策略**: 基于手牌强度和位置随机选择行动
- **适应性**: 根据筹码状况调整策略
- **多风格**: 不同AI具有不同游戏风格

### AI行动逻辑
1. 评估手牌强度
2. 计算底池赔率
3. 基于策略权重选择行动
4. 考虑当前游戏阶段
