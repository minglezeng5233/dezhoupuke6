# 德州扑克3 - 音效文件说明

## 音效文件列表

请将以下音效文件放入此目录：

### 基础音效
- `deal_card.wav` - 发牌音效
- `chip_drop.wav` - 筹码掉落音效
- `button_click.wav` - 按钮点击音效
- `shuffle.wav` - 洗牌音效

### 游戏音效
- `win.wav` - 胜利音效
- `lose.wav` - 失败音效
- `fold.wav` - 弃牌音效
- `raise.wav` - 加注音效
- `all_in.wav` - 全下音效

### 背景音乐
- `bg_music.mp3` - 背景音乐（建议使用轻快的扑克主题音乐）

## 音效要求

- 格式：WAV 或 MP3
- 采样率：44100 Hz
- 位深：16位
- 声道：单声道或立体声
- 文件大小：每个音效文件建议不超过 500KB

## 替代方案

如果没有音效文件，游戏将使用静默模式运行，不会影响游戏功能。
